# Vaani AI 🌍🎙️

**Vaani AI** is a multilingual AI-powered communication platform inspired by modern translation systems like Google Translate, built specially for Indian languages. It enables seamless **text-to-text translation** across 12 Indian languages, includes an integrated **voice reader (text-to-speech)** feature, and provides an MVP-level **voice changer module** for experimenting with voice transformation.

---

# 🚀 Features

## 🌐 Multilingual Text Translation

Translate text instantly between 12 Indian languages with high accuracy and fast response time.

### Supported Languages

* Tamil
* Telugu
* Hindi
* Kannada
* Malayalam
* Bengali
* Marathi
* Gujarati
* Punjabi
* Odia
* Urdu
* English

### Translation Capabilities

* Text-to-Text Translation
* Real-time language conversion
* Bidirectional translation support
* Unicode text handling
* Clean and responsive interface

---

## 🔊 Voice Reader (Text-to-Speech)

The built-in voice reader converts translated text into natural speech.

### Features

* AI-generated voice output
* Native pronunciation support
* Multiple language speech playback
* Adjustable speech speed
* Clear audio generation

---

## 🎭 Voice Changer (MVP)

Vaani AI also includes an experimental MVP-level voice changer module.

### MVP Features

* Basic voice modulation
* Pitch adjustment
* Male/Female voice effect
* Robot-style voice effects
* Audio preview support

> Note: This module is currently in MVP stage and under active development.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/yourusername/vaani-ai.git
cd vaani-ai
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

# ▶️ Usage

1. Open the application in browser.
2. Select source and target languages.
3. Enter text for translation.
4. Click translate.
5. Use the voice reader to hear translated output.
6. Try voice changer effects in the MVP section.
